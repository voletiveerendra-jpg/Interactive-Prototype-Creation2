# FoodieGo - Interactive Prototype

## Overview
FoodieGo is a food delivery app prototype designed in Figma. The prototype demonstrates the complete ordering process from browsing food items to confirming an order.

## Features
- Clickable navigation
- Smart Animate transitions
- Food ordering workflow
- Mobile-friendly design
- User testing and feedback

## Screens
1. Splash Screen
2. Home Screen
3. Menu Screen
4. Cart Screen
5. Order Confirmation Screen

## Tools Used
- Figma
- GitHub

## Outcome
Learned interactive prototyping, navigation flows, animations, and usability testing.
